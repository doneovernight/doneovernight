# Animations

Reserved for ordered frame strips and visual timing-review sheets.

Animation strips:

- use the state name exactly;
- order frames left to right;
- contain no labels inside the runtime strip;
- preserve the 32 x 32 cell for every frame;
- export at all approved integer scales;
- derive duration exclusively from the manifest;
- include a separate review sheet when timing annotations are required.

The Hover strip must contain exactly eight frames and exactly three jump peaks.
