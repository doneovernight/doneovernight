# States

Reserved for approved still frames representing semantic states.

Each state directory created in Phase 2 must use the controlled state vocabulary from `ASSETS/README.md` and include:

- the canonical front still;
- directional stills when the state requires them;
- reduced-motion still;
- all four approved integer-size exports;
- a state note referencing `MOTION-GUIDE.md`.

Still frames must not contain timing or transition assumptions that belong in the animation manifest.
