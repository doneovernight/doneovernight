# Turnarounds

Reserved for approved front, left, back, and right character sheets.

Every turnaround must:

- use the same 32 x 32 cell per view;
- share the `y29` ground line;
- preserve official head and body volume;
- use the official palette only;
- include no expression or prop;
- include no rear mark or seam;
- export at `@1x`, `@2x`, `@3x`, and `@8x`.
